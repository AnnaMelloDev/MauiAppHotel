# Golden Palace Hotel 🏨

Este é um aplicativo desenvolvido em .NET MAUI para a disciplina de Programação Mobile I.

## 🚀 Funcionalidades
- Cadastro de reserva de hospedagem.
- Seleção de quantidade de hóspedes e datas.
- Informações sobre a desenvolvedora (Tela Sobre).

## 🛠️ Tecnologias
- C#
- XAML
- .NET 10
